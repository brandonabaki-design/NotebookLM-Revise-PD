The Foundation ⏱️ 5 Min Read

What is EduSim V5.1?

EduSim is not a chatbot — it's an AI development team that produces a complete, state-driven web app every time you give it a topic. Imagine handing a Subject Matter Expert, Learning Designer, Simulation Architect, UI Engineer, and UAE Cultural Consultant the same mission brief — then getting one finished HTML file back.

Tiered Differentiation

UAE Context Built-In

The Architecture ⏱️ 6 Min Read

The 6-Phase Pedagogical Loop

Every EduSim output follows the same strict progression — this is what separates a real learning experience from a flashy demo. The Gem enforces it for you.

01

Main Menu

"Mission Dashboard" launches with two pathways: Learn the Theory or Enter the Simulation.

02

Tiered Theory

Interactive Field Guide with L1 / L2 / L3 toggles + dual-language TTS for accessibility.

03

Hypothesis Setup

Simulation starts paused. Students predict outcomes and set variables before hitting Initiate.

04

Active Simulation

JavaScript state machine runs — resources drain, meters fill, visuals react in real time until win/loss.

05

Level-Specific CFU

Check-For-Understanding questions are unique to the tier — DOK 1 → DOK 3/4 as rigor scales.

06

Score & Replay

Mission Complete screen combines simulation efficiency + CFU score. Play Again, Advance, or Return to Menu.

Global UI Constraint
A "Return to Main Menu" button is always visible.

Students never get locked into a screen — they can always review theory, re-attempt a tier, or change levels without reloading the app.

Differentiation ⏱️ 5 Min Read · Interactive

Tiered Scaffolding: L1 / L2 / L3

The same topic, rendered three ways. Tap a tier below to see how theory, variables, and assessment scale to match the learner.

Theory Style
Heavy Visuals · Short Sentences

Emojis, color-coded CSS diagrams, and foundational vocabulary. Example: "🌱 Plants need ☀️ sun + 💧 water to grow!"

Simulation
2 Variables · Wide Tolerance

Fewer knobs, more forgiving sweet-spots. The student can still "win" while building confidence with the mechanics.

Assessment
3 Questions · DOK 1

Identify variables, basic recall. Instant feedback reinforces correct mental models right away.

Theory Style
Standard Curriculum Depth

Clear scientific definitions, standard processes, real-world UAE application. Aligned to IB/NGSS grade expectations.

Simulation
3–4 Variables · Balanced

Realistic interdependencies — change one variable and watch the cascade. Hypothesis-driven play.

Assessment
3–4 Questions · DOK 2

"How did X affect Y?" — focuses on variable interaction and cause-and-effect reasoning.

Theory Style
Deep-Dive Scientific Literacy

Formulas, chemical/physical nuance, system-level impact analysis. Prepares for IB MYP eAssessment rigor.

Simulation
5+ Variables · Narrow Tolerance

Interlocking systems, tight sweet-spots. Students must synthesize theory before they can succeed.

Assessment
4–5 Questions · DOK 3/4

Evaluate outcomes, synthesize real-world implications, propose alternative designs.
